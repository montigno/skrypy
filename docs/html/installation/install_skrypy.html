How to install Skrypy¶

Download the installer¶

Download the latest version of the skrypy installer and save in a temporary folder.

Install¶

  1. Go to the temporary folder where the installer is located:

    cd /root/of/your/temporary/folder

  2. Unzip the installer:

    unzip skrypy_install_xx.xx.xxb.zip

  3. Launch the install:

    python3.12 setup.py /home/user/Applications/skrypy_venv

    Tip

    python3.12: choose the version of Python (see requirements chapter).
    setup.py: contains the installation code.
    /home/user/Applications/skrypy_venv: specify the installation path of your virtual environment.

    Note

    At the end of this step, 3 aliases are created in your OS’s .bashrc file.
    skrypy_312 to launch Skrypy.
    skrypy_312_install for installing Python packages.
    skrypy_312_test to test that skrypy is working properly.

  4. Reload .bashrc:

    source ~/.bashrc

  5. Install packages:

    skrypy_312_install

    Note

    This installs all packages written in Python (TensorFlow, Numpy, etc…) in your virtual environment.

    Attention

    It takes a little time and will take up about 10GB.

Test¶

  1. To test if Skrypy is working, type in the terminal:

    skrypy_3122_test
